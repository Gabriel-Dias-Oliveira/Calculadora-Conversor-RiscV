# CALCULADORA E CONVERSOR DE BASES EM RISC-V | MC404 - UNICAMP

O repositório consiste em uma calculadora e um conversor de bases (acompanhados de uma documentação) desenvolvidos para o projeto final da disciplina MC404 (Organização Básica de Computadores e Linguagem de Montagem) da UNICAMP.

## OPERAÇÕES DA CALCULADORA:
#### Operações suportadas para números decimais positivos e/ou negativos

* Adição.
* Subtração.
* Multiplicação. 
* Divisão (detecção de divisão por 0).

## OPERAÇÕES DO CONVERSOR:

* Decimal -> Binário.
* Binário -> Decimal.
* Decimal -> Hexadecimal.
* Hexadecimal -> Decimal.
* Binário -> Hexadecimal.
* Hexadecimal -> Binário.

## COMO UTILIZAR

* O código pode ser simulado usando a plataforma para a qual foi desenvolvida: [Risc-v Simulator](https://ascslab.org/research/briscv/simulator/simulator.html). 

## OBSERVAÇÕES

* O código utiliza uma versão especifica de Risc-v e foi adaptada para o simulador utilizado na disciplina 
* O exemplo de algumas simulações também pode ser encontrado no respositório (documentação). 
* Uma documentação com todas as justificativas e esclarecimento do desenvolvimento também está presente.
* Sinta-se a vontade para utilizar (e adaptar) o código.
